Weather Underground PWS API library for Node.js

# Description
`wunderground-pws` is a Node.js library that implements [Weather Underground Personal Weather Station Protocol](http://wiki.wunderground.com/index.php/PWS_-_Upload_Protocol), providing a native Javascript interface to upload weather conditions.
